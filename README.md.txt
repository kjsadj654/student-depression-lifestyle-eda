# 学生抑郁与生活方式分析（EDA 项目）

这是一个针对 10 万名大学生的生活方式、学习习惯与抑郁状态的探索性数据分析项目。

## 项目简介

使用 Python + Pandas + Seaborn 对 Kaggle 的学生生活方式数据集进行清洗、可视化和洞察挖掘，主要关注：
- 睡眠时长、学习时长、社交媒体使用与压力的关系
- 哪些因素与抑郁（Depression）相关性最强
- 性别、年龄、院系等维度的差异

## 数据来源

- 数据集：Student Depression & Lifestyle (100k Data)
- 下载链接：https://www.kaggle.com/datasets/aldinwhyudii/student-depression-and-lifestyle-100k-data
- 文件：student_lifestyle_100k.csv（原始数据约 100,000 行）
